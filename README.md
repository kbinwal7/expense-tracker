# 🐿️ SaveSavvy – Expense Tracker

SaveSavvy is a modern full-stack expense tracking application that helps users manage and monitor their daily spending with transaction history.

---

## ✨ Features

- Secure user authentication system
- Add, edit, and delete expenses
- Categorize transactions efficiently
- Search transactions by name, ID, or category
- Track transaction date and time
- Automatic expense creation timestamps
- Real-time transaction updates
- Responsive and modern UI
- FastAPI + Node.js backend integration
- PostgreSQL database support

---

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- FastAPI

### Database
- PostgreSQL
